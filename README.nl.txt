Deze app laat je een verbinding maken met een externe ssh server.
Meerdere ssh servers kunnen toegevoegd worden, waarbij voor voor ieder apparaat de inlog gegevens per apparaat worden
opgeslagen en dus niet voor iedere flow afzonderlijk hoeven te worden ingevoerd.

Bij het maken van een flow moet vanaf de SSH-Client app een commando verzonden worden waarbij in die actie kaart de te gebruiken server gekozen moet worden.

Deze app kan gebruikt worden om bijvoorbeeld:
* Een backup te laten maken
* Een spotify afspeellijst te starten.


Om zeker te zijn van een goede uitvoering van de ssh opdrachten kan een trigger flow gemaakt worden die reageert op ontvangen antwoorden en fouten.
Deze kunnen vervolgens doorgestuurd worden naar andere apps die deze meldingen loggen, of je kunt bijvoorbeeld een foutmelding notificatie naar een mobiele telefoon sturen wanneer iets niet goed is gegaan.


Opmerkingen:
Je kan geen verbinding maken met Homey zelf, die verbinding loopt altijd via Athom.
